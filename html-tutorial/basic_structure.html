To render web page correctly browsers need to recognise a HTML document by <!DOCTYPE html> tag. Here is page template:

<!DOCTYPE html>
                <html>  
    <head>
    <!-- Page title, metadata, link to CSS, etc. -->
    </head>
    <body>
    <!-- Page content -->
    </body>
    </html>
